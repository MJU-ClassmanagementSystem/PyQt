# Python client

## How to start

### 학생 등록

1. login.py 실행
2. web에서 등록한 선생님 id, password로 로그인
3. register버튼을 누르고 학생의 이름과 학번을 누르고 학생 얼굴을 등록

### 출석 체크

1. login.py 실행
2. web에서 등록한 선생님 id, password로 로그인
3. attendance버튼을 누르고 출석체크 진행

### 수업 진행

1. login.py 실행
2. web에서 등록한 선생님 id, password로 로그인
3. analyze 버튼을 누르고 해당하는 수업을 눌러서 수업을 진행
